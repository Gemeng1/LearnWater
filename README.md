# LearnWater

制作了一个接水游戏玩法粒子水是根据 render texture 渲染 裁剪了透明度低的片元 

画线 是根据LineRenderer 加Rigbody2D 和Edge Collider2D 在update 记录 鼠标的位置信息 转换成世界坐标 存储起来 进行画线
